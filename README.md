# 🚀 Summary

## 📌 Spring Security 란?
> 인증, 권한 부여 및 보호 기능을 제공하는 프레임워크 입니다.

Spring Security를 사용함으로써 인증, 권한 확인에 필요한 기능과 옵션들을 제공받을 수 있습니다.
